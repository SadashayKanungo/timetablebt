Guide

In the Schema, all Choices are grouped into Clusters. From each Cluster, a specified number of Choices are selected.
Preference is allocated in order of entry.

A Simplistic Schema :
CDCs with a single section will be merged into a single Choice in an independent Cluster of highest preference.
Courses with multiple sections will be independent Clusters, with each section as a choice. One Choice will be selected from such clusters.
HELs and DELs will be two independent Clusters, and the number of Choices to be selected for each will be specified.
